# ElevenLabs Scribe

ElevenLabs Scribe is the company's speech-to-text product line. It belongs in the speech-to-text bucket rather than the general voice bucket because the core value is transcription and structured audio-to-text conversion.

## Current status (as of 2026-04-22)

- Positioned for real-time and enterprise transcription workflows
- Relevant both for general agent pipelines and healthcare-adjacent dictation / documentation use cases
- Should not be described as universally "best" from this source set alone
