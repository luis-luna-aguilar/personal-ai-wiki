# Stitch

Stitch is Google's UI generation surface for turning prompts and references into interface artifacts. It belongs in UI generation because its value sits between design tooling and implementation scaffolding.

## Current status (as of 2026-04-22)

- Focused on generating UI concepts and interface artifacts
- Sits on the design-to-code boundary rather than being a full coding IDE
- Good example of creative and coding domains starting to overlap
