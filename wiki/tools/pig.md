# Pig

Pig is a Windows-focused computer-use platform. It is better understood as computer-use infrastructure than as a general assistant: the product centers on giving agents controlled access to Windows applications through chat, APIs, and SDK surfaces.

## Current status (as of 2026-04-22)

- Focused specifically on Windows computer-use workflows
- Exposes multiple surfaces: product UI, agent API, and lower-level SDK
- Distinctive angle is making Windows application control a programmable agent capability

## Strengths

- Clear specialization instead of vague "AI assistant" positioning
- Fits the emerging computer-use stack where operating real software is the product

## Weaknesses / caveats

- Narrower platform scope than general cross-device assistants
- Current source coverage here is still product-page level, not deep third-party usage evidence
