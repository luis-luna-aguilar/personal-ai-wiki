# Orca homepage / product overview

Homepage / landing-page overview for Orca. Frames the product as a worktree-first desktop IDE for AI coding agents: multiple isolated branches, multiple terminals, rich-file review, diff and PR/CI review, and real-time status reporting from agents like Claude Code, Codex, and OpenCode. The page is strong on product shape and workflow framing, but light on external validation.

## Key claims extracted

- Orca is positioned as "the worktree IDE for AI coding agents"
- Users can run Claude Code, Codex, OpenCode, and similar agents side by side in isolated worktrees
- Terminal workspace supports horizontal, vertical, and nested panes with restored scrollback
- Built-in editor and previews support markdown, PDFs, images, diffs, PR review, and CI/conflict triage
- Agents can report status live via the Orca CLI, including completion and blocked-state updates
- Product is available on macOS, Windows, and Linux
- Orca is open source under the MIT License
- Homepage hero advertises a built-in browser + design mode for turning UI into code
