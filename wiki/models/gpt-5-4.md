# GPT-5.4

GPT-5.4 is OpenAI's March 2026 update to the GPT-5 line. The source cluster presents it as the strongest general-purpose OpenAI model for reasoning, browsing, coding, and agent-style tasks, with especially strong benchmark positioning on knowledge work and web tasks. The same sources also note that the win is not universal: Claude still reads as stronger on writing, taste, and design-heavy work, while GPT-5.3 Codex may still have a cleaner coding style in some use cases.

## Current status (as of 2026-03-08)

- Positioned as OpenAI's latest frontier reasoning model
- Strong on reasoning, web browsing, agent workflows, and general coding
- Experimental 1M-context variant released, but the same source warns quality drops materially past roughly 256K tokens
- Pricing moved up versus prior GPT-5 releases, putting it closer to Sonnet-class economics than before
- Stronger all-around than GPT-5.3 Codex, but not always preferable for code style or design-sensitive work

## Strengths

- Broad general-purpose capability across reasoning, browsing, and agent tasks
- Lower hallucination claims than prior GPT variants in the captured sources
- Clearer "default frontier model" positioning than earlier GPT-5 iterations

## Weaknesses / caveats

- Writing style, taste, and design work still read as relative weaknesses versus Claude
- The 1M-context headline appears less practical than the marketing suggests
- Higher price weakens OpenAI's earlier cost-performance advantage

## Recent changes

- [2026-03-08] GPT-5.4 released with stronger positioning on reasoning, browsing, coding, and agent workflows
