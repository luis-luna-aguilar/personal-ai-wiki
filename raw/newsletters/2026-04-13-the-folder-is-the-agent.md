If I could summon a fleet of [AI agents](https://every.to/emails/click/647c862f45c0c6db733d6c802c5f7a124a4e4285f5493056ed476fde93921470/eyJzdWJqZWN0IjoiVGhlIEZvbGRlciBJcyB0aGUgQWdlbnQgIiwicG9zdF9pZCI6NDEwOCwicG9zdF90eXBlIjoicG9zdCIsInVybCI6Imh0dHBzOi8vZXZlcnkudG8vZ3VpZGVzL2FnZW50LW5hdGl2ZSIsInBvc2l0aW9uIjoxM30=), let them coordinate, and watch them produce work no single agent could match, it would relieve some of my overwhelm.  I tried everything to make it work—[Claude Code](https://every.to/emails/click/647c862f45c0c6db733d6c802c5f7a124a4e4285f5493056ed476fde93921470/eyJzdWJqZWN0IjoiVGhlIEZvbGRlciBJcyB0aGUgQWdlbnQgIiwicG9zdF9pZCI6NDEwOCwicG9zdF90eXBlIjoicG9zdCIsInVybCI6Imh0dHBzOi8vZXZlcnkudG8vc291cmNlLWNvZGUvaG93LWktdXNlLWNsYXVkZS1jb2RlLXRvLXNoaXAtbGlrZS1hLXRlYW0tb2YtZml2ZSIsInBvc2l0aW9uIjoxNH0=) teams, agents dispatching tasks to other agents, [orchestration setups](https://every.to/emails/click/647c862f45c0c6db733d6c802c5f7a124a4e4285f5493056ed476fde93921470/eyJzdWJqZWN0IjoiVGhlIEZvbGRlciBJcyB0aGUgQWdlbnQgIiwicG9zdF9pZCI6NDEwOCwicG9zdF90eXBlIjoicG9zdCIsInVybCI6Imh0dHBzOi8vZXZlcnkudG8vc291cmNlLWNvZGUvdGhlLXRocmVlLXdheXMtaS13b3JrLXdpdGgtbGxtcyIsInBvc2l0aW9uIjoxNX0=) where a lead agent managed a pool of workers. Many iterations, many burned tokens.  But more agents didn’t make me faster. I’ve [run parallel Claude Code sessions](https://every.to/emails/click/647c862f45c0c6db733d6c802c5f7a124a4e4285f5493056ed476fde93921470/eyJzdWJqZWN0IjoiVGhlIEZvbGRlciBJcyB0aGUgQWdlbnQgIiwicG9zdF9pZCI6NDEwOCwicG9zdF90eXBlIjoicG9zdCIsInVybCI6Imh0dHBzOi8vZXZlcnkudG8vc291cmNlLWNvZGUvaG93LWktdXNlLWNsYXVkZS1jb2RlLXRvLXNoaXAtbGlrZS1hLXRlYW0tb2YtZml2ZSIsInBvc2l0aW9uIjoxNn0=) for months, which works when each agent has a clear task, and I’m directing the work. The swarm experiment was different: agents coordinating with each other, deciding what to work on, producing output I hadn’t shaped. When 10 of them finished simultaneously, I had 10 [results to evaluate](https://every.to/emails/click/647c862f45c0c6db733d6c802c5f7a124a4e4285f5493056ed476fde93921470/eyJzdWJqZWN0IjoiVGhlIEZvbGRlciBJcyB0aGUgQWdlbnQgIiwicG9zdF9pZCI6NDEwOCwicG9zdF90eXBlIjoicG9zdCIsInVybCI6Imh0dHBzOi8vZXZlcnkudG8vc291cmNlLWNvZGUvaS1zdG9wcGVkLXJlYWRpbmctY29kZS1teS1jb2RlLXJldmlld3MtZ290LWJldHRlciIsInBvc2l0aW9uIjoxN30=) without enough context to know which ones I could trust. AI agents don’t have a speed limit, but the person managing them still does.  I kept looking for a smarter orchestration layer—a better protocol or a tighter framework that would filter the output and tell me which result to trust. Then I stopped and looked at what was really doing the work.  It was something I already had—a folder.  A project folder with a CLAUDE.md/AGENT.md (the file that tells an AI how to work in your project), some [skill](https://every.to/emails/click/647c862f45c0c6db733d6c802c5f7a124a4e4285f5493056ed476fde93921470/eyJzdWJqZWN0IjoiVGhlIEZvbGRlciBJcyB0aGUgQWdlbnQgIiwicG9zdF9pZCI6NDEwOCwicG9zdF90eXBlIjoicG9zdCIsInVybCI6Imh0dHBzOi8vc2tpbGxzLmV2ZXJ5LnRvLyIsInBvc2l0aW9uIjoxOH0=) definitions, and context accumulated through months of [compound engineering](https://every.to/emails/click/647c862f45c0c6db733d6c802c5f7a124a4e4285f5493056ed476fde93921470/eyJzdWJqZWN0IjoiVGhlIEZvbGRlciBJcyB0aGUgQWdlbnQgIiwicG9zdF9pZCI6NDEwOCwicG9zdF90eXBlIjoicG9zdCIsInVybCI6Imh0dHBzOi8vZXZlcnkudG8vc291cmNlLWNvZGUvY29tcG91bmQtZW5naW5lZXJpbmctdGhlLWRlZmluaXRpdmUtZ3VpZGUiLCJwb3NpdGlvbiI6MTl9)—that’s an agent. The context that this folder gives an AI model makes the generalized model a specialist in whatever task or field you want it to excel in.  I’m running 44 of these folders-as-agents across multiple projects now. Each one runs inside a specialized folder I’ve built and tested over months, and a dispatch layer I built on top does the routing between them. Here’s how it works.
