The command center for AI coding agents.

Orca brings the whole AI engineering workflow into one app: multiple worktrees, multiple terminals, markdown editing and viewing, PDF and image previews, diffs, PR and CI review, fast file search, and agent status tracking. It is not just a terminal for Claude Code. It is the place where you run agents, inspect output, review changes, and move work forward.

Worktree Orchestration

Spin up a worktree, drop an agent in it, and keep moving. Claude Code on auth, Codex on the API, OpenCode on the frontend with isolated branches, configurable worktree cards, and clear visibility into what each branch is doing.

feat/auth — Claude Code
fix/api — Codex
feat/ui — OpenCode
Terminal Workspace

Split terminals infinitely with horizontal, vertical, and nested panes, keep logs beside your agents, and come back to restored scrollback after restart. The terminal is a first-class workspace, not a bolt-on panel.

Docs, Notes, and Rich Files

Review product notes, specs, and agent output without leaving Orca. Edit and preview markdown, search inside markdown previews, render images in the editor, and inspect PDFs and other rich artifacts directly in your workflow.

Agents That Report Back

Agents update their status in real time via the Orca CLI: what they're working on, what they just finished, and what's blocked. Notifications and unread markers keep you from babysitting terminals all day.

File Editing, Search, and Handoff

Built-in file editing, source control, explorer actions, and quick-open across the repo, including hidden files. Jump from a diff straight to the file, copy remote URLs, and drag files directly into an agent instead of pasting paths around.
