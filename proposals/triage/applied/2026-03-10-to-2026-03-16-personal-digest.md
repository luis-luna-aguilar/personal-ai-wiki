- [x] **[coding]** Coding agents shifted from “help me write code” toward multi-agent review, delegation, and higher-level coordination.

    **What it is:** The strongest coding signal in this batch is that code-agent products are broadening from autocomplete/chat into coordinated execution. Anthropic launched Claude Code Review as a multi-agent PR review system that verifies bugs and prioritizes findings by severity, with internal testing claiming substantive PR comments jumped from 16% to 54%. Around that, JetBrains Air showed a “council of coding agents” pattern, Codex best-practice material was circulating as operational guidance rather than novelty, and Claude Code added `/btw` side conversations while work continues in the background. Put together, the center of gravity is moving toward orchestration, review, and concurrent workstreams.

    **Why it matters:** This likely updates `state-of/coding.md` and any Claude Code / Codex pages. It looks like a real workflow shift, not just another model-feature increment.

    **Sources:**
      - `raw/newsletters/2026-03-10-anthropic-drops-claude-code-review.md` — clearest launch framing for multi-agent PR review and the move from copilots toward autonomous code work

- [x] **[agents]** “Agent-native documents” emerged as a product category, with Proof positioning docs as a shared surface for humans and agents instead of laptop-bound Markdown.

    **What it is:** Every’s Proof launch is more meaningful than a generic editor launch. The core claim is that many important work artifacts are already being generated by agents, but the collaboration loop around them still happens in primitive local files. Proof reframes the document itself as a first-class agent workspace: live editing, comments, change tracking, provenance rails, and the principle that anything a human can do in the editor, an agent should be able to do too. The follow-up “Never-done Machine” digest reinforces that Every is already using it for plans, strategy docs, and daily task management.

    **Why it matters:** This looks like a strong `tools/`, `workflows/`, or `concepts/` ingest candidate around agent-native collaboration surfaces rather than just another SaaS note.
